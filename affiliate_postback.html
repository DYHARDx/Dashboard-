S2S Postback Setup

Configure your global postback URL for automated tracking

Postback Configuration

When a conversion is approved, our system will ping your URL with the following tokens:

{click_id} - Your Unique Click ID
{payout} - Conversion Payout Amount
{subid} - Optional Sub-ID / Track ID
{offer_id} - ID of the converted offer

Example: https://your-tracker.com/postback?cid={click_id}&amt={payout}

Tracking Guide

1. When promoting an offer, append your click ID to the &a= parameter.

2. Our system stores that ID and returns it via the {click_id} token upon conversion.

3. Make sure your server can handle GET requests.
